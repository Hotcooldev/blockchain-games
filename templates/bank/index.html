Game Install

The game currently only runs on Linux, and has been tested on Ubuntu 18.10.

  1. Download the code from https://github.com/kf106/blockchain-games/archive/master.zip
  2. Extract the zip file into a folder, for example ~/blockchain-games/
  3. In the folder run ./install.sh
  4. Then follow the instructions below

Game Signup

If you are running the game for the first time, start it with the following command and parameters:
./client-run.sh {{ nodeaddress }} PLACEHOLDERsignup

If you have run the game before, your blockchain node should be present, so just run the game with the command:
./client-run.sh
